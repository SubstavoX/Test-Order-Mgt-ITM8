# PROF Test Order Extension

## Beskrivelse
PROF Test Order-udvidelsen er en specialiseret løsning designet til at håndtere og administrere testordrer i Business Central. Denne udvidelse giver mulighed for at oprette, spore og administrere testordrer med særligt fokus på lot-håndtering og varestyring.

## Hovedfunktioner

### Testordrehåndtering
- Komplet administration af testordrer
- Avanceret lot-tracking system
- Integreret varestyring
- Automatiseret ordrebehandling
- Tilpassede arbejdsgange for testprocesser

### Lot Management
- Intelligent lot-sporing
- Automatisk lot-nummerering
- Detaljeret lot-historik
- Lot-specifik kvalitetskontrol

### Varefunktionalitet
- Udvidet varehåndtering
- Specialiserede varefunktioner
- Integration med eksisterende lagerstyring
- Optimeret vareflow for testordrer

## Tekniske Detaljer

### Hovedkomponenter
- TestOrder Card Page (TO_TestOrderCard_PROF)
- TestOrder Table (TO_TestOrder_PROF)
- Lot Management (TO_LotManagement_PROF)
- Item Functions (TO_ItemFunctions_PROF)
- Misc Events Handler (TO_MiscEvents_PROF)
- Single Instance Events (TO_EventsSingleInstance_PROF)

### Systemkrav
- Business Central 2022 Wave 2 eller nyere
- Compatible med både cloud og on-premise installationer

## Installation og Opsætning

### Installationskrav
1. Verificér systemkompatibilitet
2. Sikkerhedskopiér eksisterende data
3. Installér udvidelsen via Extension Management

### Konfiguration
1. Opsæt grundlæggende testordre-parametre
2. Konfigurér lot-management indstillinger
3. Tilpas vareintegration
4. Definér brugerrettigheder og -roller

## Brugerguide

### Daglig Brug
- Oprettelse af nye testordrer
- Håndtering af lot-numre
